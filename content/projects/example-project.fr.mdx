Ceci est un projet exemple pour démontrer le fonctionnement du portfolio.

## Aperçu

Les projets sont écrits en format MDX, ce qui permet d'utiliser Markdown avec des composants React.

## Fonctionnalités

- Typographie soignée avec la police Inter
- Support du mode clair/sombre
- Filtrage par catégorie
- Design responsive

## Détails Techniques

Le projet utilise Next.js 14 avec l'App Router, Tailwind CSS pour le style, et MDX pour le contenu.
