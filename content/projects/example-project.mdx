This is an example project to demonstrate how the portfolio works.

## Overview

Projects are written in MDX format, which allows you to use Markdown with React components.

## Features

- Clean typography with Inter font
- Dark/light mode support
- Category filtering
- Responsive design

## Technical Details

The project uses Next.js 14 with the App Router, Tailwind CSS for styling, and MDX for content.
